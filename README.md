# 🍽️ The Grail – Luxury Fine Dining Restaurant Website

The Grail is a modern, responsive luxury restaurant website built using HTML, CSS, and JavaScript. It delivers a premium dining experience through elegant design, smooth animations, interactive sections, and a fully responsive layout for all screen sizes.

## ✨ Features

- Elegant Hero Section
- Responsive Navigation Bar
- Light & Dark Mode
- About / Our Story Section
- Why Guests Love Us
- Signature Dishes
- Restaurant Experience Video
- Luxury Gallery
- Professional Chef Team
- Reservation Form
- Customer Testimonials
- Restaurant Statistics
- Contact Form
- Instagram Gallery
- Back to Top Button
- Scroll Progress Bar
- Smooth Scrolling Effects
- Fully Responsive Design
- Offline Font Awesome Icons Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
